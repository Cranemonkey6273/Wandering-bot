# Mobile App Assets

Use the Wandering Bot logo for the native app icon and splash screen.

Recommended source files before running `npm run assets`:

- `mobile/resources/icon.png` - 1024x1024 PNG
- `mobile/resources/splash.png` - 2732x2732 PNG

For the first build, copy `wanderingbot.png` into this folder as `icon.png`.
Replace it later with a purpose-made square icon before App Store or Google Play submission.
